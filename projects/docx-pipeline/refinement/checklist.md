# Refinement Checklist: Court Letter

## Page 1
- [ ] Letterhead image positioned correctly (top, spanning width)
- [ ] "VIA ECF" flush left below letterhead
- [ ] Address block: flush left, single-spaced, no extra gaps
- [ ] Date: right-aligned, one paragraph break below address
- [ ] RE block: proper hanging indent, consistent spacing above/below
- [ ] Salutation: same spacing as body paragraphs
- [ ] Body text: TNR 12pt, paragraph spacing consistent (~10pt after)
- [ ] First-line indent: none (block paragraph style) OR 0.5" (indent style)
- [ ] Footnote references: superscript in body text

## Continuation Pages (2+)
- [ ] Header: date + page number flush left, navy blue rule on right
- [ ] Header font: TNR 12pt (matches body)
- [ ] Header-to-body gap: ~1 blank line (200-280 twips)
- [ ] Blue rule color matches page 1 letterhead line
- [ ] Body text continues with same spacing as page 1
- [ ] No orphan headings (heading at bottom of page without body text)

## Last Page
- [ ] Signature block: closing + name + cc kept together
- [ ] Signature indented ~4" from left
- [ ] cc block flush left, directly below signature
- [ ] No excessive blank space above signature

## Footnotes (bottom of page)
- [ ] 10pt TNR
- [ ] Superscript reference numbers
- [ ] Hanging indent (number outdented, text aligned)
- [ ] Single-spaced within footnote
- [ ] Separator line above footnotes

## Global
- [ ] Margins: 1" left/right, ~1" bottom
- [ ] Footer: firm contact bar on every page
- [ ] No style bleed (e.g., heading styles affecting body text)
- [ ] Compare side-by-side with precedent PNGs
